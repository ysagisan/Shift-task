# Shift-task

### **Требования к запуску:**
- _Версия Java_: 21  
- _Система сборки_: Maven 4.0.0  

### Сборка проекта
_Перейдите в корень проект и выполните_  

` mvn clean package`  

_После успешной сборки исполняемый файл будет создан в директории:_  
`target/`  

### Запуск приложения  
_общий формат запуска:_  
`java -jar target/util-1.0.jar [опции] <входные_файлы>` 

_пример запуска:_  
`java -jar target/util-1.0.jar -p sample- -f in1.txt in2.txt`  

### Опции
| Опция         | Описание                                   |
| ------------- | ------------------------------------------ |
| `-a`          | Режим добавления в выходные файлы  |
| `-s`          | Вывод краткой статистики                   |
| `-f`          | Вывод полной статистики                    |
| `-p <prefix>` | Префикс для выходных файлов                |
| `-o <path>`   | Директория для выходных файлов             |
